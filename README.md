Структура проекта

- `lab3/` — ООП-версия с тестами
- `lab4/` — файловая СУБД
  - `in_memory_repository.py` — хранение в оперативной памяти
  - `file_repository.py` — хранение в JSON файле
  - `tui.py` — текстовый интерфейс
- `run_lab4.py` — точка входа с выбором типа хранилища

Инструкция по запуску

1. Запустите `python run_lab4.py`
2. Выберите тип хранилища (1 или 2)
3. Используйте меню для работы с книгами
